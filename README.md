# Chess Game in Flutter

## Overview

This is a fully-functional chess game built using Flutter, designed for both beginners and experienced players. The game allows you to play against another player locally, with intuitive UI and smooth animations.

## Features

- Local Multiplayer: Play against a friend on the same device.
- Chessboard: Interactive chessboard with drag-and-drop functionality.
- Move Validation: Enforces standard chess rules, ensuring valid moves.
- Undo Moves: Ability to undo the last move.
- Game States: Displays check, checkmate, and stalemate conditions.
- Piece Promotion: Supports pawn promotion with a selection dialog.
- Responsive Design: Optimized for both mobile and tablet screens.

## How to Play
- Start a Game: Open the app and press the "New Game" button to start.
- Make a Move: Drag a piece from its current position to a valid square.
- Undo Move: Press the "Undo" button to take back the last move.
- Check/Checkmate/Stalemate: The app will automatically detect and display these conditions.
